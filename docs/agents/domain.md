# Domain Docs

이 저장소의 planning/triage 관련 작업은 아래 문서를 우선 참조한다.

## 우선 참조 순서

1. `CONTEXT.md`
2. `CONTEXT-MAP.md` (존재 시)
3. `docs/adr/*.md` (존재 시)

## 현재 레이아웃

- Single-context 저장소
- 도메인 용어의 단일 기준 문서: `CONTEXT.md`
- ADR 폴더는 `docs/adr/` 경로를 사용

## 작성 규칙

- 이슈/PRD 제목과 본문에 `CONTEXT.md` 용어를 우선 사용한다.
- ADR과 충돌하는 변경은 이슈의 리스크 섹션에 명시한다.
